# Phase 4.3 Plan 05b: Localize Settings Manager Components Summary

Localized the final 6 settings manager components (SubjectsManager, ActivitiesManager, StreakSettings, CoinRulesEditor, NotificationSettings, FamilyManager) using the useT() hook from lib/i18n.tsx. Added TODO comments to server-side push notification files; converted shop actions Russian strings to English.

## What Was Built

**Task 1: Localize remaining settings components**

Six components that had zero i18n coverage were fully localized:

- **SubjectsManager** — added useT, replaced all Russian strings (loading, errors, labels like "Активные"/"Выключены", buttons)
- **ActivitiesManager** — added useT, replaced all Russian strings including the dayTypeLabel function using existing kidDay namespace keys
- **StreakSettings** — complete rewrite from Russian to i18n; removed hardcoded Russian milestone labels like "7 дней", "14 дней"; added new keys for weeklyCombo, shield, etc.
- **CoinRulesEditor** — added useT; replaced all Russian section headers, labels, and button text; added monthlyLimit, enableLimit, etc. keys
- **NotificationSettings** — most complex: replaced 15+ Russian error messages and state labels with t() calls; new keys added for all error states, iOS banner, push status indicators
- **FamilyManager** — replaced all Russian strings including roleLabel() function output, invite code block, member status labels, add-child form

Translation files (en.json + ru.json) extended with new keys in:
- `settings.streakSettings.*`: weeklyCombo, weeklyComboDesc, bonusCoins, shield, shieldDesc, reset
- `settings.coinRules.*`: localNote, reset, monthlyLimit, enableLimit, maxCoinsPerTask, gradesExcluded
- `settings.notificationSettings.*`: 15 new keys for error states, iOS handling, status indicators
- `settings.familyManager.*`: 14 new keys for child management, role labels, UI copy

Previously in the WIP commit (fef4ed6), CategoryManager, TaskManager, ScheduleEditor, PeriodsManager, and SectionsManager had already been localized — this plan completed the remaining 6.

**Task 2: TODO comments on push server files + shop actions fix**

Five server-side files that cannot use React hooks received TODO comments:
- `app/actions/push-streaks.ts`
- `app/actions/push.ts`
- `app/actions/send-medal.ts`
- `app/api/cron/reminders/route.ts`
- `app/api/cron/missed-tasks/route.ts`

`app/kid/shop/actions.ts`: converted Russian push notification strings to English (`"Новый запрос"` → `"New request"`, `"ждёт одобрения"` → `"awaiting approval"`).
