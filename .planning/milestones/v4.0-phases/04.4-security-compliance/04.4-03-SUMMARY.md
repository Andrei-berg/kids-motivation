## What Was Built

**AuditScreen** (`components/parent-center/screens/AuditScreen.tsx`):
- Props: `{ familyId: string; children: ParentChild[] }`
- Fetches from `getAuditEvents()` with `limit: 30` per page
- Child filter pills: "All" + one button per child, re-fetches on change
- Event row: emoji icon by action_type + description + `date-fns` formatted timestamp + coins delta (green/red)
- Empty state + 3-row shimmer skeleton on load
- "Load more" button appends next page (offset pagination)
- All strings via `useT()` under `audit.*` namespace

**ParentCenter routing**:
- Route type extended: `... | 'audit'`
- `case 'audit': return <AuditScreen familyId={familyId ?? ''} children={children}/>`
- Nav item added: `{ id: 'audit', label: t('audit.navLabel'), icon: 'tasks' }` (6th item in bottom nav)

**i18n keys added** to both `en.json` and `ru.json`:
- `audit.title`, `audit.navLabel`, `audit.filterAll`, `audit.loadMore`, `audit.noEvents`, `audit.loading`
