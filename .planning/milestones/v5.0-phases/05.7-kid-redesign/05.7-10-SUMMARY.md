## Decisions Made
- Preserved the `triggerGoalConfetti` export rather than deleting it, since
  the out-of-scope dead-code `components/GoalsModal.tsx` still imports and
  calls it — deleting the export would fail `npx tsc --noEmit`. Deletion is
  explicitly deferred to that file's dead-code disposition per the plan.
- Used a sentinel-split technique (`t('kidChallenges.reward', { reward: ' ' }).split(' ')`)
  to interleave the gold `Amount` atom inside the existing "Reward: {{reward}}"
  translation string without adding a new i18n key or hardcoding coin values
  in the component.
