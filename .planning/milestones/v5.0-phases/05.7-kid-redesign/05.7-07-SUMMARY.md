## Decisions Made
- **Balance strip removed:** the old sun-gradient "ТВОИ МОНЕТЫ" strip duplicated what ScreenHeader now provides (gold Amount balance) and was built entirely from `T.sun`/`T.sunDeep` legacy tones — removing it is part of the reskin, not scope creep.
- **KMButton retired from this page:** `tone="coral"`/`tone="teal"` props are legacy color names in touched markup; replaced with plain indigo (`paper.accent`) CTA buttons per the color contract (indigo = interactive).
- **Touch targets:** all touched buttons raised to `minHeight: 44` per the binding UX-02/UI-SPEC invariant (previously 34px).
- **Child name source for ScreenHeader:** `api.getChild(activeMemberId)` (added to the existing `Promise.all` load, `.catch(() => null)`); the `Child` model has no avatar field, so `avatarUrl` is omitted and the Avatar fallback renders.
- **Stamp mount point:** the ApprovalSheet's timed `approved` stage is the page's approval-celebration moment (no confetti previously existed); the staged sent→seen→approved logic is unchanged per CLAUDE.md's preserve-logic rule.
