## User Setup Required

Set `CRON_SECRET` environment variable:
- In `.env.local` for local testing: `CRON_SECRET=any-secret-string`
- In Vercel project settings: Dashboard → Settings → Environment Variables → `CRON_SECRET`

Vercel Cron automatically sends `Authorization: Bearer $CRON_SECRET` in production.
To test locally: `curl -H "Authorization: Bearer your_cron_secret" http://localhost:3000/api/cron/reminders`
