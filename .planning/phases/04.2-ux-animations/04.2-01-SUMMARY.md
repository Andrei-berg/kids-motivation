## Decisions Made
- `mode="wait"` on AnimatePresence: exit completes before next page enters, preventing content overlap that would cause its own visual flash
- `initial={false}`: skips the mount animation on hard reload — only cross-route navigations get the transition
- Scale goes 0.98→1 (not 0→1) for a subtle depth effect without being jarring
- OfflineBanner/PushInit/InstallPrompt stay outside PageTransition: they are persistent UI elements that should never animate out on navigation
- Framer Motion v11 natively reads `prefers-reduced-motion` — no extra CSS or conditional logic needed for accessibility
