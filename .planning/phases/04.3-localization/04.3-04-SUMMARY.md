## Decisions Made

- coachRatingModal.* keys committed atomically with CoachRatingModal.tsx — no window where component could render without keys
- RATINGS array defined inside component function body so t() calls work at render time (not module load time, which is before LanguageProvider mounts)
- Extended key groups (auditLogExtra, walletDashboardExtra, etc.) used for strings beyond the initial namespace plan — avoids nesting conflicts
