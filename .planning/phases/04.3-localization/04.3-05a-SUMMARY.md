# Phase 04.3 Plan 05a: Wallboard + Parent Shop + Onboarding + 5 Components i18n Summary

One-liner: Applied `useT()` i18n to wallboard page, parent shop, both onboarding pages, auth page, PotentialWidget, ExpenseModal, ExpenseCharts, and ScheduleEditor — replacing all hardcoded Russian UI strings with translation keys.

## What Was Built

All 9 client files now use `useT()` from `lib/i18n.tsx`. Translation keys were already defined in both `en.json` and `ru.json` for most strings (from plan 04.3-01). Missing `onboarding.*` keys were added to `ru.json` (60+ keys), and a new `wallboard.streakBonusText` key was added to both locale files.

## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 2 - Missing] CATEGORY_LABELS and ROLES arrays moved inside components**
- **Found during:** Task 1 — editing parent/shop/page.tsx and onboarding/join/page.tsx
- **Issue:** Module-level const arrays (`CATEGORY_LABELS`, `ROLES`) referenced Russian strings but couldn't call `useT()` at module scope
- **Fix:** Moved both arrays inside the component function body where `t` is available
- **Files modified:** app/parent/shop/page.tsx, app/onboarding/join/page.tsx

**2. [Rule 2 - Missing] DAYS array in ScheduleEditor moved inside component**
- **Found during:** Task 1 — editing ScheduleEditor.tsx
- **Issue:** Same pattern — `DAYS` array with Russian day names at module level
- **Fix:** Moved inside `ScheduleEditor` component body
- **Files modified:** components/ScheduleEditor.tsx

**3. [Rule 1 - Bug] Tab variable naming conflict in app/page.tsx**
- **Found during:** Task 1 — editing app/page.tsx
- **Issue:** The map callback used `t` as the loop variable, shadowing the `t` translation function
- **Fix:** Renamed loop variable to `tabKey`
- **Files modified:** app/page.tsx

**4. [Rule 2 - Missing] ru.json onboarding section had only 12 keys; en.json had 72**
- **Found during:** Task 1 — after completing all file edits
- **Issue:** plan 04.3-01 pre-populated en.json with all onboarding keys but left ru.json minimal

- **Files modified:** public/locales/ru.json

**5. [Rule 1 - Bug] wallboard.tsx had `bonusText` with hardcoded '7 дней → +100₽' strings**
- **Found during:** Task 1 — grep scan after completing main edits

- **Files modified:** app/wallboard/page.tsx, public/locales/en.json, public/locales/ru.json
