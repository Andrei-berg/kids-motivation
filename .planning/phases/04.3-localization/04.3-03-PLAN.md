
Translation key namespaces (from public/locales/en.json):
- achievements.* — achievements page strings
- kidWallet.* — kid wallet page strings
- kidShop.* — kid shop page strings
- kidDay.* — kid day page strings
- kidLeaderboard.* — leaderboard page strings
- kidDayFill.* — KidDayFillForm section labels
- common.loading, common.coins, common.days — shared strings

Variable interpolation syntax: t('key', { varName: value })
Examples:
  t('achievements.badgesCount', { count: allBadges.length }) → "/ 42 badges"
  t('achievements.level', { level: 3 }) → "LVL. 3"
  t('kidWallet.earnedThisWeek', { amount: weekScore }) → "+15 this week"
  t('kidLeaderboard.untilSunday', { days: 3, dayWord: t('kidLeaderboard.days') }) → "⏱️ Until Sunday: 3 days"
