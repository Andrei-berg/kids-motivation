- Props: `{ familyId: string; currentMemberId: string; senderName: string; senderRole: 'parent' | 'child' }`
- State: `messages: ChatMessage[]`, `loading: boolean`

  - system messages: centered italic gray text, no bubble

- handleSend: calls `sendMessage({ familyId, senderId: currentMemberId, senderName, senderRole, messageType: 'text', content: text })`. Add optimistic message to state immediately before awaiting; on error show console.warn (do not crash).
