<interfaces>
Key contracts the tests build against (verified this session):

From app/api/wallet/award/route.ts:
- `POST(req: NextRequest)` — body `{ childId, date }` (date `YYYY-MM-DD`); responds `{ ok, creditedCoins, awards }`.
- Source types emitted (singular): `room` (sourceId = days.id), `behavior` (days.id, parent-role only), `grade` (subject_grades.id), `sport` (section_visits.id), `activity` (activity_logs.id), `book` (reading_log.id), `streak` (sourceId = date string).
- Auth chain: `requireFamilyMember()` (cookie session — MUST be mocked in tests) → `createAdminClient()` (service-role — real) → `assertChildInFamily` (real).
- Streak bonus: reads global `streaks` (current_count thresholds 7/14/7 for room/study/sport) and global `settings` table keys `roomStreak7|studyStreak14|sportStreak7`, default 100.

From app/api/wallet/_lib.ts:
- `creditAwards` pre-checks `wallet_transactions` on `(child_id, source_type, source_id)`, applies deltas via `wallet_apply` RPC, unique index 23505 is the race backstop.
- `loadSettings` falls back to SETTINGS_DEFAULTS (grade5=10, grade4=5, grade3=-3, room=3, behavior=5, coach5=10, book=20).

From lib/supabase/admin.ts:
- `Membership = { userId, familyId, role: 'parent'|'child'|'extended', childId: string|null }`.
- `createAdminClient()` needs `SUPABASE_URL` (or NEXT_PUBLIC_) + `SUPABASE_SERVICE_ROLE_KEY`; throws otherwise.
- CAUTION: admin.ts imports `./server` which imports `next/headers` — the vi.mock of '@/lib/supabase/admin' must be a partial mock (importActual) overriding ONLY requireFamilyMember/requireParent, or if next/headers breaks module load under vitest, mock '@/lib/supabase/server' with a stub instead.
</interfaces>

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-051-13 | Information Disclosure | Service-role key in test output | mitigate | Key read only from env; never logged/asserted; test output contains ids and balances only |
| T-051-14 | Repudiation | Leftover test rows polluting analytics | mitigate | afterAll teardown deletes all seeded rows + family; final assertion checks zero remaining wallet_transactions for the test child |
| T-051-15 | Denial of Service | CI/dev without keys | mitigate | describe.skipIf(hasIntegrationEnv) keeps `npm test` green with no env |
</threat_model>
