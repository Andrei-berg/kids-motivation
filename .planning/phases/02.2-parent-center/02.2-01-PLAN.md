3. **Skeleton loading**: While data loads, render 2-3 skeleton cards (`animate-pulse bg-gray-800 rounded-xl h-32`) — not blank screen (REQ-UX-002)

4. **Per-child status card** (dark theme, `bg-gray-800 rounded-xl p-4`):
   - Child emoji + name as card header (`text-xl font-bold text-white`)
   - Coins earned this week: `💰 {weekScore.total} монет`
   - Room status indicator: green dot if `dayData?.room_ok`, gray dot otherwise
   - Behavior indicator: green checkmark if `dayData?.good_behavior`, gray otherwise
   - Grades: number of graded days this week (`gradedDays`)
   - "Заполнить день" button → navigates to `/parent/daily?childId={child.id}`

7. **Page title**: `<h1 className="text-2xl font-bold text-white">Сегодня</h1>` at top with current date subtitle
