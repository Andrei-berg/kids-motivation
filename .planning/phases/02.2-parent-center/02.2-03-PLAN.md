**Panel 2 — P2P Transfer** (only visible if family has 2+ children):
- Section title: `Перевод между детьми`
- Source child: already selected (current tab)
- Target child dropdown: `<select>` listing other children by name
- Amount input (`type="number"`, min=1)
- Submit button: `Перевести` — calls `createP2PTransfer({ fromChildId: selectedChildId, toChildId, coins: amount, description: 'Перевод от родителя' })`
