<objective>
Establish one source of design truth and wire the three brand fonts, WITHOUT changing any
consuming component. `lib/design/tokens.ts` holds the base palette (indigo, gold, success,
warning, danger) and two themes — `paper` (kid) and `ink` (parent). The two existing `T`
objects become thin re-exports that keep every key they have today but resolve to values
from the new system, so `tsc` alone proves the recolor breaks no consumer. The three brand
fonts (Bitter / Golos Text / JetBrains Mono) load via `next/font/google` with cyrillic
subsets and expose CSS variables that the token font strings reference.

Purpose: this is the foundation both later plans (atoms, pilots) build on, and it is the
only plan that recolors every kid + parent-center screen at once (via token substitution).
Output: lib/design/tokens.ts, recolored kid/parent token re-exports, font-wired layout.
</objective>
