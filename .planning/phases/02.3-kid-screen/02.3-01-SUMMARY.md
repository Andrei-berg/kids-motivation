## What Was Built

### KidNav (components/kid/KidNav.tsx)
- `'use client'` component using `usePathname()` for active tab detection
- 5 tabs: Мой день (violet), Кошелёк (amber), Достижения (emerald), Магазин (rose), Рейтинг (cyan)
- Mobile: `fixed bottom-0` bar with `env(safe-area-inset-bottom)` safe area via `kid-nav-bottom` class
- Active tab: per-tab colored icon + label + bg pill on icon
- Desktop: `sticky top-0 h-16` bar with horizontal tab links
- `aria-label` on every tab link and nav element

### KidLayout (app/kid/layout.tsx)
- Server component (no 'use client')
- `bg-amber-50` wrapper for all `/kid/*` pages
- `pb-20 md:pb-0 md:pt-16` on `<main>` to clear mobile bottom bar / desktop top bar

### /kid redirect (app/kid/page.tsx)
- Replaced broken localStorage-based client component
- Server-side `redirect('/kid/day')` — no client JS, no localStorage

### Kid CSS (app/globals.css)
- Appended "KID MODE STYLES" section — zero modifications to existing CSS
- 6 gradient variables: `--kid-gradient-hero` through `--kid-gradient-shop`
- 3 shadow variables: `--kid-shadow`, `--kid-shadow-hover`, `--kid-shadow-glow`
- Utility classes: `kid-card`, `kid-hero-gradient`, `kid-coin-gradient`, `kid-xp-gradient`, `kid-nav-bottom`
- Animation classes: `kid-skeleton`, `kid-xp-bar`, `kid-fire-pulse`, `kid-fire-glow`, `kid-float-coin`
- Event classes: `kid-celebration-overlay`, `kid-celebration-badge`, `kid-coin-bounce`, `kid-bottom-sheet`
- State classes: `kid-badge-locked`, `kid-goal-glow`

### Badge Catalogue (lib/services/badges.service.ts)
- BADGES array extended from 6 to 10 entries
- New: `coin_saver` (300 XP), `first_purchase` (200 XP), `streak_30` (700 XP), `full_week_grades` (350 XP)
- `checkCoinSaver`: wallet.total_earned_coins - total_spent_coins >= 500
- `checkFirstPurchase`: reward_purchases count >= 1 for child
- `checkStreak30`: any streaks row with current_count >= 30
- `checkFullWeekGrades`: 5+ unique grade dates within current week range
- `checkAndAwardBadges` now calls all 8 check functions (4 original + 4 new)
