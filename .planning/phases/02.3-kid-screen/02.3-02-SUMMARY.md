## Decisions Made

- **getWeekData return shape:** The actual return is `{ days: DayData[], grades: SubjectGrade[], sports, weekRecord }` — not the aggregate `Week` type. Week stats computed directly from per-day/grade arrays, giving accurate roomDays, behaviorDays, and gradedDays counts.
- **XP bar formula:** `(child.xp % 1000) / 10` gives progress within current level cycle (0–100%)
- **Streak fire thresholds:** >5 days → `kid-fire-pulse`, >10 days additionally `kid-fire-glow` — matches spec
- **Grade average color:** 3-tier threshold matching parent UI (≥4.5 emerald, ≥3.5 blue, else amber)
