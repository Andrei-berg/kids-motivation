**State:**

- `mood`: string | null — initialise from `existingDay.mood`
- `checkedActivities`: Set<string> — activity IDs the child checked
- `activities`: ExtraActivity[] — loaded from `getExtraActivitiesByDayType`
- `settings`: WalletSettings | null
- `saving`: boolean

**Live coin calculation** (computed, no state):
```typescript
const coinsPreview = useMemo(() => {
  let total = 0
  const roomCount = Object.values(roomItems).filter(Boolean).length
  if (roomCount >= 3 && fillMode >= 2) total += settings?.coins_per_room_task ?? 3
  checkedActivities.forEach(id => {
    const act = activities.find(a => a.id === id)
    if (act) total += act.coins
  })
  return total
}, [roomItems, checkedActivities, activities, settings, fillMode])
```

**Lock logic:**
```typescript
const isLocked = useMemo(() => {
  const today = new Date().toISOString().slice(0, 10)
  return date < today
}, [date])
```

**useEffect on mount:** Load `getExtraActivitiesByDayType(childId, dayType)` and `getWalletSettings()` in parallel.

**handleSubmit:**
1. Set `saving = true`
2. Build saveDay params based on fillMode:
   - Mode 1: only `mood`, `noteChild` (diary note if any)
   - Mode 2: room items + `mood`
   - Mode 3: room items + `mood`
   Always include `filledBy: 'child'`
3. Call `saveDay({ childId, date, ...roomParams, filledBy: 'child', mood })`
4. If fillMode >= 2 and room_ok (≥3 items), call `awardCoinsForRoom(childId)`
5. For each checked extra activity, call `logActivity(childId, date, activityId, true)` and `updateWalletCoins(childId, act.coins, activityName, act.emoji)`
6. Call `onSaved(coinsPreview)`

**Note on `updateWalletCoins`:** Import directly from `lib/repositories/wallet.repo.ts` — it's already exported.
