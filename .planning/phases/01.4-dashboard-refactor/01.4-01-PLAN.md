<task type="auto">
  <name>Task 1: Create useFamilyMembers hook</name>
  <files>lib/hooks/useFamilyMembers.ts</files>
  <action>
    Create the directory lib/hooks/ if it does not exist, then create lib/hooks/useFamilyMembers.ts.

    The hook must:
    1. Be marked 'use client' at the top
    2. Export interface FamilyMember { id: string; display_name: string; avatar_url: string | null; role: 'parent' | 'child' | 'extended' }
    3. Export function useFamilyMembers() returning { members: FamilyMember[], loading: boolean, error: string | null }
    4. On mount (useEffect with [] deps), call createClient() from '@/lib/supabase/client'
    5. Call supabase.auth.getUser() — if no user, set loading=false and return early (middleware handles redirect)
    6. Query family_members table: .select('family_id, id, role').eq('user_id', user.id).maybeSingle() to get the caller's membership row
    7. If no membership found, set loading=false and return early
    8. Write family_id to store: if membership.family_id !== familyId, call setFamilyId(membership.family_id)
    9. Query all children: .from('family_members').select('id, display_name, avatar_url, role').eq('family_id', membership.family_id).eq('role', 'child').order('display_name')
    10. Role-based filtering (REQ-FAM-010 — child sees only themselves): after step 9, check if membership.role === 'child'. If so, filter the fetched children array to only include the entry whose id === membership.id. This ensures a child user's NavBar shows only their own pill button and cannot switch to a sibling's data.
    11. Set members state from the (potentially filtered) children array
    12. Auto-select logic: if activeMemberId is null OR if activeMemberId is not in the fetched members array (stale localStorage), set activeMemberId to members[0].id (if at least one child exists)
    13. Wrap everything in try/catch, set error state on catch
    14. Always set loading=false in finally

    Import pattern: use { useEffect, useState } from 'react'; { createClient } from '@/lib/supabase/client'; { useAppStore } from '@/lib/store'

    Note: The hook runs on every page because NavBar is inside each page (not in layout). This is acceptable — 2 simple Supabase queries per page navigation for a family app with 2-3 children.
  </action>

  </verify>
