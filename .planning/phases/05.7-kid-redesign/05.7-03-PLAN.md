## STRIDE Threat Register
| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-057-06 | Tampering | chat_last_read_at UPDATE | mitigate | UPDATE RLS `USING/WITH CHECK (user_id = auth.uid())` — a member can mark only their own row's read state, never another member's. |
| T-057-07 | Information Disclosure | getChatUnreadCount | mitigate | Reads only within the child's own `family_id` under existing chat_messages SELECT RLS; no cross-family leakage. |
