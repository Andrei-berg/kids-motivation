## What Was Built

### Wallet page (D-16, D-13, D-15)
- Hand-rolled gradient/glass hero (old lines 96-152) replaced with `<ScreenHeader title={t('kidHeader.wallet')} coins name/>` + a paper-card hero block: `useCountUp(wallet.coins)` fed into `<Amount theme="paper" money size="xl"/>` — no bespoke gold inline number.
- Weekly summary line beneath the hero: `+earned` as a signed gold `Amount` (earned = existing `getWeekScore` total) `/` `−spent` as a neutral `Amount` (spent = sum of negative `coins_change` among the already-loaded transactions within the current week). Render-only; no new money computation.
- Transaction history keeps the 05.3-03 `LedgerRow theme="paper"` pilot verbatim; empty state now uses `wallet.emptyHeading`/`wallet.emptyBody`.
- Local `useDesktop` copy deleted; shared `lib/hooks/useDesktop` imported. Desktop 340px sticky goals column (4.5 layout) preserved.
- Child name for ScreenHeader fetched via `api.getChild(activeMemberId).catch(() => null)` (day-page pattern); `Child` model has no avatar field so the Avatar SVG fallback renders.
- Dropped with the old hero: the ≈-equivalent line and BALANCE/SAVED footer chips (part of the replaced block); quick actions grid untouched.

### GoalsPanel (D-16, D-20)
- `triggerGoalConfetti()` call and `utils/confetti` import removed. Completion celebration modal is now a paper-tone card with a `<Stamp trigger={justDone.id}>`-wrapped `StampPlaque` — bordered, uppercase `t('stamp.goalReached')` («ЦЕЛЬ ДОСТИГНУТА»), −8° tilt, 450ms entrance via the Stamp atom's built-in reduced-motion double guard (no third guard added).
- In-progress goal card: current/target render as gold `Amount` (md/sm) figures replacing raw `toLocaleString` strings; ProgressRing on paper tones (`paper.accent` progress, `paper.successText` when reached, `paper.lineSoft` track).
- Completed card: teal gradient retired for `paper.card` + `paper.success` border with a small tilted stamp plaque; CTA buttons use `paper.accent`.
- Create/edit sheet behavior unchanged (no delete action, per UI-SPEC).

### Amount atom
- `size` union extended with `'xl'`; sizes map gains `xl: { fs: 30, fw: 800 }` with a comment marking it hero-only (atom-owned type exception).
