# Milestone 5 — Flexibility & Design Unification: «Гибкость»

**Цель:** Ни одно правило дня не зашито в код — всё настраивается семьёй. Один бренд
(FamilyCoins) и одна дизайн-система («семейный банк»: бумага/чернила) на весь продукт.

**Принцип:** ничего не перекрашиваем, пока оно зашито. Сначала данные — потом дизайн.

**Дизайн-контракт:** https://claude.ai/code/artifact/ab9621cc-2f84-42ff-a873-d07f8b841715
(13 мокапов, палитра indigo #6C5CE7 + монетное золото #E9A83C, шрифты Bitter / Golos Text /
JetBrains Mono, подпись бренда — строка гроссбуха с отточием).

## Фазы

| # | Фаза | Суть |
|---|------|------|
| 5.1 | launch-prep | Ротация ключей, Sentry+аналитика, интеграционные тесты денег, имя FamilyCoins |
| 5.2 | room-tasks | Чеклист комнаты из таблиц `room_tasks`/`room_checks` вместо 5 колонок схемы |
| 5.3 | design-tokens | Единый tokens.ts (темы «бумага»/«чернила») + общие атомы (LedgerRow, Amount) |
| 5.4 | streak-settings | Пороги/бонусы стриков из констант award → `wallet_settings` |
| 5.5 | year-calendar | Учебный год, региональные пресеты каникул, выходные семьи, «болезнь» |
| 5.6 | day-blocks | Сборка дня: тип дня × расписание × правила блоков; фиче-флаг на семью |
| 5.7 | kid-redesign | Kid UI на единой системе, навигация 6→5 |
| 5.8 | parent-redesign | Parent Center + Конструктор дня + Календарь года + Итог недели |
| 5.9 | rules-presets | Пресеты правил (Классика/Без штрафов/Только бонусы), grade_scale, теги поведения |
| 5.10 | automation | Карманные по расписанию, автоодобрение до лимита, умные напоминания |
| 5.11 | legacy-cleanup | Снос 10 легаси-страниц + globals.css, брендинг иконки/сплэша |

## Ключевые инварианты

- Миграции только аддитивные; переходы через двойную запись; легаси сносится последним.
- Деньги: все мутации server-side (service-role), идемпотентность `(child_id, source_type, source_id)`,
  verify-скрипты (`scripts/verify-*.mjs`) должны проходить после каждой фазы, трогающей award.
- 5.6 (day-blocks) выкатывается только за пер-семейным фиче-флагом.
