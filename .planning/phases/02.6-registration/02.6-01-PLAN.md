```typescript
function generateChildId(name: string): string {
  const slug = name.toLowerCase().replace(/[^a-z0-9]/g, '_').replace(/_+/g, '_').replace(/^_|_$/g, '').slice(0, 12)
  const base = slug || 'child'
  const suffix = Math.random().toString(36).slice(2, 6).padEnd(4, '0')
  return `${base}_${suffix}`.slice(0, 20)
}
```

Constraint: result must match `/^[a-z0-9][a-z0-9_]{1,18}[a-z0-9]$/` (no leading/trailing underscores, 3–20 chars). The `.replace(/^_|_$/, '')` before `.slice(0, 12)` handles that for the slug portion; `base` fallback handles empty names.
