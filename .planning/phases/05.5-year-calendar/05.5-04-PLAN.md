| user input (dates/region) → DB | Free-form date and region strings entered in the form. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05.5-11 | Tampering | term_mode / weekend_days input | mitigate | DB `CHECK (term_mode IN ('quarters','trimesters'))` (from 05.5-01) plus client-side toggles that can only emit valid weekday indices bound the values; malformed input is rejected at the DB. |
| T-05.5-12 | Elevation of Privilege | family_calendar write | mitigate | upsert scoped by `family_id`; RLS INSERT/UPDATE policies confine writes to the caller's family (05.5-01). A parent cannot edit another family's calendar. |
| T-05.5-13 | Tampering | year_start/year_end ordering | accept | Client-side validation blocks `year_start > year_end`; a bypassed bad range only mis-classifies that family's own days (no cross-family or money impact). |
| T-05.5-SC | Tampering | npm/pip/cargo installs | mitigate | No package installs in this plan; N/A. |
</threat_model>
