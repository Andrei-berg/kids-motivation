## User Setup Required

**VAPID keys and email require manual configuration.** Before push notifications will work:

1. Generate VAPID key pair:
   ```bash
   npx web-push generate-vapid-keys
   ```

2. Add to `.env.local`:
   ```
   NEXT_PUBLIC_VAPID_PUBLIC_KEY=<Public Key from above>
   VAPID_PRIVATE_KEY=<Private Key from above>
   VAPID_MAILTO=mailto:admin@kidscoins.app
   ```

3. Add same env vars to Vercel project settings:
   - Vercel Dashboard → kids-motivation project → Settings → Environment Variables
   - Add all three variables

4. Optionally add `CRON_SECRET` to protect the /api/push/send endpoint from unauthorized calls.
