**Create app/settings/page.tsx** — FULL REPLACEMENT of existing file.

The page is a 'use client' component. Structure:

1. **PIN gate** — on mount, if not authenticated, show full-screen PIN entry. Use existing `verifyPin()` from `@/utils/helpers`. After correct PIN, show settings. Show error message after wrong PIN.

2. **familyId loading** — on mount (after PIN passes), call:
   ```typescript
   const supabase = createClient()
   const { data: { user } } = await supabase.auth.getUser()
   const { data: member } = await supabase
     .from('family_members')
     .select('family_id')
     .eq('user_id', user.id)
     .maybeSingle()
   setFamilyId(member?.family_id ?? null)
   ```

3. **Section navigation** — vertical list of section cards in a sidebar (or top tabs on mobile). Sections: "Категории", "Задачи", "Монеты", "Стрики", "Расписание", "Уведомления". Active section highlighted with indigo border.

4. **Content area** — renders the active section's component, passing `familyId`.

5. **NavBar** — rendered at the top (consistent with other pages). Import from `@/components/NavBar`.

6. **Styling** — dark background (#111827 or Tailwind `bg-gray-900`), white/gray text, indigo/emerald accents. Cards with `bg-gray-800 rounded-2xl p-6`. Mobile-first. Max width `max-w-2xl mx-auto`. Clean, minimal, polished — not crammed.

PIN gate UI: centered card, "Настройки родителя" title, "Введите PIN-код" subtitle, 4 large digit buttons (0-9 grid), display of entered digits as ●●●●, "Войти" button. Same style as the rest of the app.

Phase 1.3 scope: This component displays the configurable coin rules as an informational/editable UI. In this phase, it shows the grade coin values and allows the parent to see and update them. The actual coin engine enforcement happens in Phase 2.1; this phase stores the values in a simple approach.

Implementation approach for Phase 1.3: Store grade coin rules as a JSONB config on the family row, OR simply display the hardcoded defaults with an "editable" UI that saves to localStorage for now (since the engine isn't wired yet).   | 2       | -50    |
