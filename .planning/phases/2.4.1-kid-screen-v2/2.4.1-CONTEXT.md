<decisions>
## Implementation Decisions

### Kid Day-Fill Form — Scope (configurable per family)

Parent configures in settings which mode is active:
- **Mode 1:** Mood + extra activities only (room and grades = parent only)
- **Mode 2:** Room + mood + extra activities (grades = parent only)
- **Mode 3:** Everything except grades (room, mood, sport, extra activities)

All 3 modes must work — the setting switches behavior, not separate screens.

### Day Type Detection

Auto-detect from date:
- Mon–Fri → school day
- Sat–Sun → weekend
- Any date inside `vacation_periods` table → vacation

Form shows activities relevant to day type (extra_activities already has `day_types: ['vacation', 'weekend']` field). On school days show school-related activities; on vacation/weekend show vacation/weekend activities from ActivitiesManager config.

### Coins on Submit

Coins credited **immediately** when kid submits the form. Parent can apply correction/penalty later if dishonesty detected.

### Trust System

The system is built on trust. If child is dishonest → parent applies a manual penalty (existing correction mechanism). No automated verification — honesty is the contract.

### Form Layout

Sections layout (scroll):
- 🏠 **Комната** — checklist (tap to toggle each item, shows +coins per item)
- 🌟 **Настроение** — emoji picker (😄 😐 😔 😠 😢)
- ⚽ **Доп. занятия** — checklist from ActivitiesManager, filtered by day type
- Each section shows coins earned so far

Kid can see what parent already filled (grades, parent notes) as read-only.

### Editability

Kid can re-edit their submission **until end of day (00:00)**. After midnight, locked.

### Weekly Calendar

7-day horizontal strip showing the current week:
- Each day cell: day letter (П/В/С...) + coin total for that day
- Today highlighted
- Filled days show actual coins (green if positive, amber if low)
- Unfilled days show placeholder/empty state
- Tapping a past day → show summary (read-only), not edit form
- Show coins earned live as kid fills today

### Expenses Tab

Kid sees expenses spent on them by the family:
- List of sections (sports clubs) with monthly cost
- Total per month spent on the child
- Simple summary view, not detailed transactions
- Framing: "Семья тратит на тебя" — motivates appreciation

### Navigation & Layout

Tabs inside `/kid/day` page (not separate routes):
- **Сегодня** — fill form + today summary
- **Неделя** — weekly calendar strip + week total
- **Расходы** — expenses summary

Bottom tab nav already exists in `/kid/layout.tsx` — day, wallet, shop, achievements. No new nav items needed; tabs live inside the day page.

Design: bright, gamified — gradients, coin animations on submit, emoji-rich. Top-tier mobile UX.

### Claude's Discretion

- Exact coin animation on submit
- Skeleton loading states
- Empty state illustrations
- Color palette for day cells (earned/not earned/today)

</decisions>
