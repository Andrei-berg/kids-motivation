# ONBOARDING.md — Первое знакомство с приложением

> Цель: новая семья за 3 минуты настраивает всё и делает первую запись. Ни одного лишнего шага.

---

## Принципы онбординга

1. **Прогресс виден** — progress bar сверху (шаг 1 из 5)
2. **Назад всегда можно** — нет тупиков
3. **Skip разрешён** для некритичных шагов (добавить бабушку можно потом)
4. **Ничего не теряется** — данные сохраняются при каждом шаге
5. **Тон: дружелюбный, не корпоративный** — "Как зовут вашего ребёнка?" не "Введите имя пользователя"

## Онбординг ребёнка (отдельный флоу)

Когда ребёнок скачивает приложение и вводит код семьи:

### R1 — Вход ребёнка
```
"Привет! Ты новый участник семьи Бергов?"
[Да, это я!]

Выбери своего персонажа:
[Адам 12 лет 🧒]    [Алим 8 лет 👦]

Как ты хочешь войти?
[Создать пароль]  [Войти через Google]
```

### R2 — Добро пожаловать в твой мир
```
"Привет, Адам! 👋

Вот как это работает:
⭐ Выполняешь задания → получаешь монеты 🪙
💰 Копишь монеты → покупаешь что хочешь
🔥 Каждый день → стрик растёт
🏆 Достигаешь целей → получаешь бейджи

Готов начать зарабатывать?"

[Погнали! 💪]
```

## Ключевые правила онбординга

| Правило | Описание |
|---|---|
| Не спрашивай лишнего | Email, имя, дети — этого достаточно для старта |
| Дай попробовать сразу | После онбординга — сразу на дашборд, не в туториал |
| Не пугай настройками | Расписание, магазин — в отдельном разделе настроек |
| Прогресс-бар обязателен | Человек должен видеть что он не застрял |
| Отмечай завершение шага | Чекмарки, анимации — каждый шаг должен "засчитываться" |
