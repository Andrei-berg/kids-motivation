# Current Milestone: M2 — Core Loop

**Goal:** Children can earn coins, spend in shop, see streaks/badges/analytics.

## Success Criteria
1. Coin rules configurable by parent (DB-backed, not hardcoded)
2. Child wallet shows balance + transaction history
3. Shop: parent creates items, child purchases, parent approves
4. Badges auto-awarded when conditions met
5. Analytics: weekly coins chart, grade distribution, KPIs
